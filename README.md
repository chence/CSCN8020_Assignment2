## Project summary

This project implements Q-Learning in the Taxi Gymnasium environment. The taxi agent learns how to move, pick up a passenger, and drop the passenger at the correct destination. The notebook includes the Q-Learning algorithm, object-oriented Python classes, training metrics, plots, hyperparameter experiments, a best-combination experiment, and a final evaluation.

## How to run

1. Clone the repository:

```bash
git clone https://github.com/chence/CSCN8020_Assignment2.git
cd CSCN8020_Assignment2
```

2. Create and activate a virtual environment:


## Q-Learning implementation

The implementation uses a Q-table with 500 states and 6 actions. The agent selects actions using an epsilon-greedy strategy. After each transition, it updates the Q-value using:

```text
Q(S,A) <- Q(S,A) + alpha * [R + gamma * max Q(S',a) - Q(S,A)]
```

The notebook tests the baseline hyperparameters and then changes the learning rate and exploration factor separately. Finally, it selects a best parameter combination and runs one more training experiment.
